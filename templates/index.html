Display Data

  • Displays the top 10 rows of any table in the database.
  • Shows the table statistics

Model Results

  • Runs the neural network model on the table selected in the drop-down list box.

ChatGPT Analysis

  • Note that the ChatGPT API does not remember previous queries. They are independent.
  • Submits the results of a model to ChatGPT for analysis.
  • Returns the analysis with an explanation of the model.

ChatGPT Conversation

  • Contains a text box and a Submit to ChatGPT button.
  • Allows a conversation with ChatGPT.
